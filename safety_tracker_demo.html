The field app operators use to log inspections. Every record carries mandatory photo evidence and writes to a single shared database with an automatic audit trail.
Safety Equipment—
Three scheduled workflows run the governance loop automatically — no one has to chase the data. Recipients shown are generic roles.
Flow 01 · Daily
Daily Health Summary
⏱ Recurrence · 06:00 daily
  • Read all asset records from the database
  • Aggregate Healthy vs Unhealthy counts
  • Flag any new deficiencies in the last 24h
  • Build a formatted HTML summary table
  • Email the digest to the site team
→ Site Team gets a morning health digest, every day.
Flow 02 · Compliance
Overdue Audit Check
⏱ Recurrence · 07:00 daily
  • Read all records, check last-inspected date
  • Filter assets past their audit cycle (Mon/Thu, Tue/Fri, Wed/Sat rotation)
  • Condition: any overdue?
  • If yes → build overdue list
  • Email the responsible shift supervisor
→ Shift Supervisor is nudged before a slot is missed.
Flow 03 · Escalation
72-Hour Management Escalation
⏱ Recurrence · checks state
  • Find assets stuck in Unhealthy > 72 hours
  • Build a hardened escalation table
  • Compose the high-priority notice
  • Email Section In-Charge + Management
→ Nothing unsafe sits unresolved >72h without management seeing it.
